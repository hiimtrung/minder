# Phase 6 Tracker — Branch Topology, IDE Bootstrap, and Secure Installer Automation

**Goal**: move CLI-side branch-topology discovery plus repository-local IDE bootstrap, secure MCP/instruction/agent installation, and cross-platform installer automation into a dedicated post-Phase-5 track so Minder can provision repo-scoped LLM integration without diluting the current transition focus on cross-repo impact traversal.

## Status

| Area                          | Status        | Notes                                                                  |
| ----------------------------- | ------------- | ---------------------------------------------------------------------- |
| Phase 6                       | `NOT STARTED` | Queued after `Phase 5`; intentionally separated from `AT-T11`          |
| Repo-local IDE bootstrap      | `BACKLOG`     | Minder CLI must install MCP, instructions, and agents per repository   |
| Secure installer distribution | `BACKLOG`     | Quick-install scripts and versioned updates planned for GitHub release |

## Scope Note

This phase carries forward the old transition backlog item `AT-T10` and extends the follow-up work into repository-local IDE bootstrap and installer automation.

The transition tracker has already closed runtime search, impact, and prompt-context follow-up in `AT-T11`, while this phase owns the remaining CLI/repo-scanner automation for discovering and submitting branch topology, provisioning repo-scoped IDE integrations, and distributing secure install/update flows.
